# Bitcoin Price Monitoring

Uma interface moderna e minimalista para monitoramento do preço do Bitcoin em tempo real, utilizando taxas de câmbio para Real (BRL), Dólar (USD) e Euro (EUR).

## Funcionalidades
- Consumo de dados em tempo real via API da Binance.
- Interface gráfica moderna com suporte a temas (CustomTkinter).
- Formatação automática de moedas.

### Tecnologias Utilizadas
- **Python 3**
- **CustomTkinter**: Para uma interface de usuário moderna.
- **Requests**: Para integração com APIs REST.
- **Pillow (PIL)**: Para manipulação de ícones e imagens.
